Установка Qwik Messenger PWA

Пошаговая инструкция

Инструкция по установке

1 Скачайте архив с приложением

Скачайте файл qwik-messenger-pwa.zip с сайта.

Скачать ZIP архив

2 Создайте папку для приложения

Создайте папку на вашем компьютере, например C:\QwikMessenger, куда будет установлено приложение.

3 Распакуйте архив

Распакуйте содержимое скачанного ZIP-архива в созданную папку.

  • Щелкните правой кнопкой мыши по скачанному файлу
  • Выберите "Извлечь все..."
  • Укажите путь к созданной папке
  • Нажмите "Извлечь"

4 Запустите PowerShell как администратор

Откройте PowerShell с правами администратора:

  • Нажмите Win+X и выберите "Windows PowerShell (администратор)"
  • Подтвердите запрос на повышение прав

5 Разрешите выполнение скриптов

Для работы приложения необходимо разрешить выполнение скриптов. Введите команду:

Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser

При запросе подтверждения введите Y и нажмите Enter.

6 Перейдите в папку приложения

В PowerShell перейдите в папку, где находится приложение:

cd C:\QwikMessenger

(замените путь на тот, который вы использовали в шаге 2)

7 Запустите приложение

Запустите приложение с помощью скрипта:

.\start-app.ps1

Если все сделано правильно, скрипт запустит локальный сервер.

8 Откройте приложение в браузере

Откройте браузер и перейдите по адресу http://localhost:3000

9 Установите приложение как PWA

В правой части адресной строки должна появиться иконка установки (на разных браузерах может выглядеть по-разному).

  • Нажмите на иконку установки
  • Подтвердите установку
  • Теперь приложение установлено на вашем компьютере и доступно через меню Пуск

Примечание: Офлайн-режим

Приложение поддерживает офлайн-режим работы. Если вы отправляете сообщения без подключения к интернету, они будут автоматически синхронизированы, когда подключение восстановится.

Системные требования:

  • Windows 10 или новее
  • Node.js: https://nodejs.org/
  • MongoDB: https://www.mongodb.com/try/download/community
